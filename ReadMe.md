# Projeto gerenciador de tarefas

Este repositório contém o projeto python que cria um gerenciador de tarefas.

Comandos normalmente utilizados na rotina de versionamento de código:

git init: Inicia um repositório .git NOVO (Cria a pasta .git e um ramo (branch) padrão (geralmente master ou main)).
git branch -M "main": Você pode, por meio desse comando, mudar o nome da branch principal para main.

git add . : Todas as modificações feitas nas pastas onde estão seus códigos, são PREPARADAS PARA SEREM incluídas no repositório (área de standing).

git commit -m "NOME_DO_COMMIT": Cria o seu commit 

git add 'NOME_DO_ARQUIVO.type': Coloca seu arquivo na área de standing.

git status: 

git remote add origin <insira aqui o endereço web do seu repositório> : Conexão do repositório remoto com o repositório local (Máquina para web).


git chekhout -b 'NOME_DA_BRANCH':

git push -u origin main : Faz subir suas novas atualizações no site do GitHub de fato.

Git clone: Pode copiar um repositório remoto (disponível no github) para o seu computador.

ls: listar os arquivos de um repositório

cd: mover para dentro de um arquivo
